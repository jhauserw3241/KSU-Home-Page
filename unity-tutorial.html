Description
This tutorial is a live demo to show how to create a basic 2D side scroller game in Unity. This tutorial is mainly focused on beginners or people who haven't used Unity before.

Prerequitisites
  1. Install Visual Studio(Any version should work)
  2. Install Unity version 5.x from
  3. Download the images from the "Necessary Images" section
  4. Put the images in a folder on your desktop
